# Initiative Tracker

## What is this for?

Tracking initiative and HP for characters during the game. 

For more info, open [docs.html](https://stormreaever.github.io/dnd-initiative-tracker/docs.html)

## Technical Stuff

### Stuff this uses

- Sass
- AngularJS
- Gulp

### Installation

1. run npm install
2. open index.html in ya browser
